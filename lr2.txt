Списки управления доступом (ACL)

ACL прав доступа - определяет доступ к файлу или каталогу на основе установленных прав
ACL по умолчанию - назначается только папке. Все объекты внутри такой папки наследуют ACL родительской папки

Модификация прав доступа осуществляется командой setfacl -m
Просмотр прав доступа - командой getfacl <имя объекта>

Пример установки ACL:(Установка разрешение и )
student@alexander-VirtualBox:~$ sudo chmod 755 StickyBit/
student@alexander-VirtualBox:~$ ls -l
итого 44
drwxrwxr-x  6 student student 4096 авг 30 14:19  real
drwx------  6 student student 4096 сен 24 13:11  snap
drwxr-xr-x+ 3 student student 4096 ноя  6 09:20  StickyBit
student@alexander-VirtualBox:~$ getfacl StickyBit/
# file: StickyBit/

Установка разрешения для пользователя tmp
student@alexander-VirtualBox:~$ getfacl StickyBit/
# file: StickyBit/
# owner: student
# group: student
# flags: --t
user::rwx
user:tmp3:r-x
user:tmp4:rwx
group::rwx
group:tempgrp:rwx
mask::rwx
other::rwx

Установка разрешения для группы
student@alexander-VirtualBox:~$ setfacl -m g:tempgrp:rwx StickyBit/
student@alexander-VirtualBox:~$ getfacl StickyBit/
# file: StickyBit/
# owner: student
# group: student
# flags: --t
user::rwx
user:tmp3:r-x
user:tmp4:rwx
group::rwx
group:tempgrp:rwx
mask::rwx
other::rwx

При смене разрешений командой chmod ACL сохрвняются в прежднем виде, но не сбрасывается mask
student@alexander-VirtualBox:~$ sudo chmod 755 StickyBit/
student@alexander-VirtualBox:~$ ls -l
student@alexander-VirtualBox:~$ getfacl StickyBit/
# file: StickyBit/
# owner: student
# group: student
user::rwx
user:tmp3:r-x
user:tmp4:rwx			#effective:r-x
group::rwx			#effective:r-x
group:tempgrp:rwx		#effective:r-x
mask::r-x
other::r-x
:wq
